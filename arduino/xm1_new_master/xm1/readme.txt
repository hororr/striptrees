You have to install the folowing libraries for arduino:

Adafruit_NeoPixel
https://github.com/adafruit/Adafruit_NeoPixel.git
Clone to: /arduino/libraries
Or add Neopixel in library manager of arduino.


Esp8266
https://github.com/esp8266/Arduino.git
Clone to: /arduino/hardware/esp8266com

Or follow the manual:
http://www.wemos.cc/wiki/doku.php?id=en:arduino_get_started
